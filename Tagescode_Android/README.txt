Tagescode Android-Widget

Funktionen:
- Zeigt auf dem Startbildschirm nur den aktuellen Tagescode an.
- Aktualisiert sich bei Datumswechsel, Zeitänderung, Zeitzonenänderung und Neustart.
- Unterstützt automatisch den Hell- und Dunkelmodus.
- Tippen auf den Tagescode öffnet die Datumsauswahl.
- In der Datumsauswahl kann ein anderes Datum gewählt und dessen Tagescode angezeigt werden.
- Sobald die Datumsauswahl geschlossen oder verlassen wird, zeigt das Widget sofort wieder den heutigen Tagescode.

APK mit Android Studio erstellen:
1. Projektordner in Android Studio öffnen.
2. Gradle-Synchronisierung abwarten.
3. Build > Build App Bundles or APKs > Build APKs wählen.
4. Die Datei liegt danach normalerweise unter:
   app/build/outputs/apk/debug/app-debug.apk
